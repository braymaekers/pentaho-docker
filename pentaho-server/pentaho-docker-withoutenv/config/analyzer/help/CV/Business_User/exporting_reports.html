Export to PDF and Save or Print

Complete the following steps to print a report.

  1. Open the report you want to print.
  2. Select More actions and options » Export » To PDF.
  3. The Export to PDF settings dialog displays: Choose the Page Format and Orientation for the export..
  4. Click Export to generate the PDF (This will convert your report into a PDF file (Adobe Acrobat) and launch a new window) or click Done to simply save the export settings for future exports.
  5. Use the Adobe Print command (File » Print) or Save As command (File » Save As) in this window.

Export to Excel with Formatting

Complete the following steps to export an Analyzer report to Excel:

  1. Open the report you want to print.
  2. Select More actions and options » Export » To Excel. (This will convert your report into a .xlsx file (Microsoft Excel) and launch a new window) or click Done to simply save the export settings for future exports.

Export Report Data to CSV

Complete the following steps to download report data into CSV format.

  1. Open the report whose data you want to download.
  2. Select More actions and options » Export » To CSV.
  3. In the Export to CSV dialog, choose if you want to Include Subtotals or Keep Measure Formatting, then click Export (This will open an Excel sheet displaying your report data).

Note: When you download data in the CSV format you will get numbers with the full precision available. This way you avoid any rounding errors when you continue to work with your data in Excel.

By default, your data will be saved in CSV (Comma Separated Values) format. If you'd like you can complete the following steps to convert the data from CSV format into a regular Excel file (XLS).

  1. Select File » Save As in Excel to open the Save As dialog box.
  2. Select Microsoft Excel (*.xlsx) from the Save as type menu.
  3. Click the Save button to save the file in XLSX format.

Note: Only the data, with no report formatting, is transferred when you Export to CSV.
